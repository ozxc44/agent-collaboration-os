Get Started in 3 Minutes See Features
# 3 minutes to your first agent event stream
from zz_agent import ZZClient

client = ZZClient(api_key="<jwt-or-zzk-key>")
project = client.create_project("My Team")
agent = client.register_agent(project.id, "Assistant")
session = client.create_session(project.id, [agent.id])
for event in client.stream_events(session.id):
    print(event.type, event.payload)

Project Spaces

Multi-agent workspaces with RBAC, shared sessions, and full event traceability.

  • Role-based access (owner / admin / member / viewer)
  • Multi-agent sessions with append-only event log
  • Session versioning & optimistic concurrency
  • Webhook outbound with HMAC signing & retries
📊

Health Monitor

Datadog-grade observability for every agent — heartbeats, metrics, alerts, and incident timelines.

  • Real-time agent health (heartbeat, uptime, success rate)
  • Model metrics (latency, tokens, cost)
  • Built-in alert rules (down, failure rate, lag)
  • Incident lifecycle tracking
